# fugue ☕️

just start typing.

you have 10 seconds to change what you wrote.

<img src="resources/readme_images/demo.gif">

just keep typing. don't worry about making it perfect.

let your ideas flow freely. when you're done, press `⌘ + S` to save your work as a text file, or `⌘ + P` to print.

<a href="https://hunterirving.github.io/fugue/">try it now</a> in your web browser, or install fugue as a <a href="https://hunterirving.github.io/web_workshop/pages/pwa/">Progressive Web App</a> that works completely offline.
